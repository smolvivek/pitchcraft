# PRICING.md — PitchCraft Pricing Strategy

**Status:** Approved (Feb 2026)
**Rule:** Claude does NOT change pricing without explicit creator approval.

## Three Tiers + 14-Day Pro Trial + Flat Commission + Metered AI

---

### Free — $0/forever

| | |
|-|-|
| Active pitches | **1** |
| Sections | **All** (8 required + 18 optional) |
| Media uploads | Images (10MB each), PDFs (50MB each) |
| Video embeds | **Yes** (YouTube, Vimeo, any URL) |
| Share link | **Public only** |
| AI | **None** |
| Version history | **Current version only** (no history) |
| Funding | **Yes — 8% commission** |
| Pitch slug | Random UUID |
| Branding | "Made with Pitchcraft" footer |
| View notifications | None |
| Collaborators | None |
| PDF export | None |
| Custom sections | None |

**What gates the upgrade (6 real triggers):**
1. **Second pitch** — new project, can't create it
2. **Privacy** — can't send privately or password-protect
3. **AI** — text assist and image generation buttons visible but locked
4. **Custom slug** — UUID looks unprofessional when sharing verbally
5. **Branding** — "Made with Pitchcraft" footer on a pitch sent to a studio exec
6. **Commission** — 8% vs 5% on funded amounts (saves $150 per $5K raised)

**Cost to PitchCraft per free user: ~$0** (1 pitch, no AI, negligible)

**14-day Pro trial** included on signup. Creator experiences full Pro features, then drops to Free if they don't subscribe. No credit card required for trial.

## 14-Day Pro Trial

Every new signup gets 14 days of Pro features. No credit card required.

**Why:** Creators need to experience AI, privacy controls, and unlimited pitches to understand why Pro is worth $12/month. Without a trial, conversion depends on imagination. With a trial, it depends on experience.

**After trial expires:** Account drops to Free tier. All pitches remain accessible but only 1 can be edited. Private/password links revert to public. AI features lock. No data is lost.

---

## Funding Commission — Flat Per Tier

| Tier | Commission |
|------|-----------|
| Free | **8%** |
| Pro | **5%** |
| Studio | **3%** |

*Stripe's 2.9% + $0.30 per transaction is always separate.*

Flat rates. No graduated breakpoints. Easy to understand, easy to communicate.

- Free at 8% is still better than Kickstarter (8-10% effective)
- Pro at 5% matches Kickstarter but with a permanent link, not a temporary campaign
- Studio at 3% is best-in-class for any platform with built-in funding
